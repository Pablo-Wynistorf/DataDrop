Quick Start

# Login to DataDrop

datadrop login --api

# Upload a file

datadrop upload myfile.txt

# List your files

datadrop list

# Get share URL

datadrop get-url --name myfile.txt

Commands

loginAuthenticate with DataDrop
logoutRemove stored credentials
statusShow login status
uploadUpload a file
